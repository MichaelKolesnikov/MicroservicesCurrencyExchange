# Лабораторная №5
Колесников Михаил Леонидович

Приложение обмена валют, реализованное в микросервисной архитектуре.

Список сервисов:
- spring-cloud-config-server
- service-registry
- gateway
- user-service
- payment-service

Настроены grafana (tempo, loki, prometheus)

Используется СУБД PostgreSQL
